## Dev (개발 메모)
모듈 다운 

`npm install`

실행 

`node app.js`

#### views
 `views/layouts` : 레이아웃 (반복 코드 미리 구성)
 `views/components` : 부품 (header, footer 등등)
 `views/business` : 실제 로직 

## 프로젝트
[EON](https://github.com/kyechan99/EON-WebServer) 프로젝트의 프론트엔드 진행중 괜찮은 아이디어가 떠올라 분리개발하기로 결정.
